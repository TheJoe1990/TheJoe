Current Version: 1.0

Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://thejoe1990.github.io/repo/ (Give it the name "The Joe").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "The Joe Repository", and install repository.thejoe-1.0.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "The Joe1990 Repo"
  • Go into the "Repository add-ons" section in the repo, and you'll find The Joe1990

Enjoy!
